## 1. 产品定位

ThisIsMyQuant 是一款国内期货与 A 股的本地市场发现、模拟盘、本地数据库和引用式 AI 分析桌面应用，定位为 **本地化、可解释、面向练习、研究与复盘的金融市场工作台**。它帮助用户完成：

```
市场发现 → 标的详情 → 自选 → 基础模拟盘 → 事件资讯 → 本地数据 → 引用式 AI
```

项目提供虚拟资金和本地模拟撮合，不连接期货公司或券商交易柜台，不做实盘下单，不托管真实账户。所有行情缓存、资讯、报告、模拟订单、成交、持仓、资金曲线、A 股数据和复盘记录优先保存在本地 SQLite。

## 2. 市场与品种范围

| 板块 | 代表主力品种 | 说明 |
|---|---|---|
| 黑色建材 | 螺纹钢、热卷、铁矿石、焦炭、焦煤、玻璃、纯碱 | 地产、基建、钢厂利润、铁水、港口库存。 |
| 有色贵金属 | 沪铜、沪铝、沪锌、沪镍、氧化铝、碳酸锂、黄金、白银 | 美元利率、库存、冶炼利润、新能源、避险。 |
| 农产品软商品 | 豆粕、豆油、棕榈油、玉米、白糖、棉花、苹果、生猪 | 天气、进口、压榨、库存消费比、养殖利润。 |

v1 不补充金融期货。股指、国债等金融期货仅作为后续扩展项，不进入当前默认品种目录、首页统计、批量分析和报告工作流。

A 股发现入口已合并到**市场**页面，通过资产类型筛选和搜索统一访问。

## 3. 目标用户

| 用户角色 | 核心诉求 |
|---|---|
| 个人期货练习者 | 用虚拟资金练习国内期货下单、持仓和复盘。 |
| 个人期货交易者 | 快速看清主力品种行情、异动原因、模拟持仓风险和次日关注点。 |
| 基本面研究用户 | 把资讯、日历、库存、供需、政策等因素归入可追溯框架。 |
| 量化/开发用户 | 本地可扩展数据链路、模拟交易数据库、SQLite 留痕、可替换 LLM Provider。 |
| A 股研究用户 | 需要指数、行业、个股、财报、资金流、筛选器和模拟组合复盘。 |
| 自托管用户 | 凭据本地保存，数据不上传到项目服务器。 |

## 5. 非功能需求

| 类别 | 要求 |
|---|---|
| 性能 | 首页聚合接口首屏可用；K 线 1 万根查询目标 < 200ms。 |
| 稳定性 | 数据源失败可降级，错误状态必须可见，不阻塞主界面。 |
| 安全 | 凭据不进日志，不进入 LLM Prompt，不提交 Git。 |
| 合规 | 仅提供模拟交易；不连接实盘柜台、不发送真实委托；抓取类数据仅用于个人研究和本地分析，不商业转售。 |
| 可观测 | 关键任务具有状态、更新时间、错误信息和 trace 日志。 |
| 可测试 | 前端 mock E2E、Rust 集成测试和 live client E2E 覆盖核心链路。 |

## 7. 范围界定

### In Scope

- 国内商品期货与航运运价主力数据。
- A 股指数、行业/概念、个股、财报、资金面、公告新闻和模拟组合。
- AKShare/Sina 行情、金十资讯/日历、Yahoo 外盘参考。
- AKShare、Baostock、Tushare Pro 可选的 A 股数据源。
- 本地 SQLite 存储和 Tauri 桌面端。
- 虚拟账户、基础模拟下单、持仓、资金、风控、绩效分析。
- 事件资讯聚合、报告、引用式 AI 分析、异动快评。
- 五大板块的专业化 UI 与因子框架。

### Out of Scope

- 实盘交易、真实下单、期货公司账户登录、交易密码保存、银期转账。
- 券商实盘交易、证券账户登录、交易密码保存、真实证券委托。
- v1 金融期货默认覆盖。
- 移动端原生 App。
- 多用户 SaaS 账号体系。
- 高频 Tick 交易系统或完整策略回测平台。
- 以主入口形式提供止损/止盈/OCO/移动止损/条件单、历史行情回放训练（这些仅作为设置页实验区功能）。

## 9. 验收标准

每个功能模块完成需满足：

1. 前端 TypeScript、ESLint、单元测试无 error。

3. Mock E2E 覆盖页面可访问、主要操作和空/错状态。
4. 涉及真实链路时，client live E2E 覆盖健康检查、专业工作台和报告免责声明。
5. 文档同步更新，包括需求、架构、通信、数据源、使用说明。

## 10. 术语表

| 术语 | 含义 |
|---|---|
| 主力连续 | 用 `RB0`、`AU0` 等表示的主力合约连续行情。 |
| 因子 | 影响期货价格的可解释变量，如库存、供需、宏观、政策。 |
| 决策流 | 将行情、资讯、日历、异动和报告串联的时间线。 |
| 数据质量 | 对数据新鲜度、来源和置信度的状态标识。 |
| CTP | 国内期货主流交易/行情接口，v1 暂不直接接入。 |
| 模拟盘 | 使用虚拟资金和本地撮合练习交易，不产生真实委托。 |
| 撮合引擎 | 根据行情和订单规则生成模拟成交的本地服务。 |
| LLM | 大语言模型，用于研报、摘要和追问。 |
| A 股 | 中国内地上市股票市场，本文默认指沪深北 A 股普通股与主要指数。 |
| 实验区 | 设置页中的进阶功能区域，承载复杂订单、复盘、回放等，不作为主导航入口。 |
